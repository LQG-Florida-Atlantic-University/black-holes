# black-holes


"BH_Gaussian_github.nb": includes Mathematica codes of the improved Hamiltonian, effective equations of motion (EOMs) and changes of variables, numerial solution of effective EOMs, plots of Kretschmann invariant, analyze of dS_2 x S^2 geometry, and classical limit of effective EOMs.

"BH_Gaussian_EOM.jl": Julia code of numerical solution of effective EOMs, plots of Kretschmann invariant, and plots of numerical error.

"BH_Gaussian_EOM.ipynb": Jubyter notebook of the Julia code.
